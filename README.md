# News-Breeze
 News Breeze is an un-official news App which gets latest news.

# Modules 
1. Home ( Including Splash )
2. Current News 
3. Saved News

# Tech Stack Used

1. MVVM
2. Retrofit
3. Dependency Injection(Dagger2)
4. Glide
5. Mutable Live Data
6. Unit Test(Done for POJO class and started for viewHolder)
